Raspberry Pi Smart Mirror



The project is very simple to recreate, and involves the use of a two-way mirror like this one, a Raspberry Pi single-board computer, and an old monitor. The project relies upon MagicMirror – an open source smart mirror platform available on GitHub. There are plenty of good tutorials online demonstrating how to make your own, so I highly recommend giving this project a go – especially if you have an old, unused monitor lying around which could be put to good use.
